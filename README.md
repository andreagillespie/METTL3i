# METTL3i
Scripts for analysis conducted as part of Guirguis et al manuscript: https://aacrjournals.org/cancerdiscovery/article-abstract/13/10/2228/729345/Inhibition-of-METTL3-Results-in-a-Cell-Intrinsic?redirectedFrom=fulltext Experiments included in this repo are:

RNAseq_220329: Initially processed and analysed by YCC, these scripts are for secondary analysis I conducted.

SLAMseq_220203: B16-F10 mice treated w/ DMSO, drug 2457 or drug 3006 w/ 4sU, 4sU_U or none, 9 samples total, n=1.

SLAMseq_220622: AT3 mice treated w/ DMSO, drug 2457 or drug 3006 all +/- interferon; w/ 4sU_30min, 4sU_60min, 4sU_6hr, 4sU_U_6hr or none + T0. All n=3, 94 samples in total.
